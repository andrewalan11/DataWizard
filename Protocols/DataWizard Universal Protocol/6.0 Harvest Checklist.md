## 7. End-of-Harvest Checklist

A harvest session is not complete until all three steps are done. If context runs out, note the incomplete state in the session log and flag to the human.

1. **Source YAML** -- set `harvest_status`, `harvested_into` (with section-level anchors), `harvest_date`, `harvest_notes`. Or `harvest_status: reviewed` with `harvest_notes` explaining why nothing was harvested.
2. **Harvest Ledger** -- add or update row in `0.4 Harvest Ledger - [Project].md`. Include source, harvest date, destinations, and agent.
3. **Session log** -- add entry to `0.2 Session Log.md` as part of normal session logging.
