
# DataWizard

**AI-assisted knowledge management for Obsidian.** Give this page to Claude (or any AI assistant) and it will walk you through setting up everything from scratch.

For the full quickstart guide, see [Protocols/DataWizard Quickstart.md](Protocols/DataWizard%20Quickstart.md).

---

## Quick Setup

1. **Install Obsidian** — https://obsidian.md
2. **Download this repo** — Click **Code → Download ZIP**, unzip into `_DataWizard/Seed/` in your vault
3. **Connect Claude to Obsidian** — See `Guides for Humans/Connecting Obsidian to Claude Desktop on Mac.md`
4. **Create a Claude Project** — Paste the instructions from `PASTE INTO CLAUDE.md` (in the Seed root)
5. **Start a conversation** — Claude will orient itself and help you set everything up

For detailed instructions, give the [Quickstart guide](Protocols/DataWizard%20Quickstart.md) to Claude and it will walk you through everything step by step.
